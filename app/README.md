## 说文解字

multimodal/multilingual Chinese dictionary powered by AI
            
It is built on [Streamlit](https://streamlit.io/) web framework ([Source](https://github.com/wgong/zinets)) : 

## How to use

```
streamlit run ZiNets.py
```

## 字子 - 说文解字 2.0

multimodal/multilingual Chinese dictionary powered by AI，built on [Streamlit](https://streamlit.io/) web framework ([Source](https://github.com/wgong/zinets)) : 

## How to use

```
streamlit run ZiZi.py
```
